Live Odds Fetch

This page fetches live odds data from the Parions Sport API on load and logs the response to the console. The data can be used to analyze hedging strategies for sports betting bonuses.

Note: The API call may be blocked by CORS. If you see a CORS error in the console, consider using a proxy server (e.g., hosted on Vercel) to bypass CORS restrictions. Check the console for the API response.
